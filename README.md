# J2EE Project

## Description
It is a project that uses Hibernate mapping Single Table Inheritance to register users, such as students or employers.

## Functionalities

* Single Table Inheritance.

* JSP's and Servlets.

* ORM Hibernate Framework.

* CRUD through Hibernate.

## Features
* Manage User Info.
